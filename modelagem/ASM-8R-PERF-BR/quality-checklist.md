# Checklist de Qualidade

## Bloqueio inicial

- [ ] Silhueta geral confere com Serie 8R acima de 2020.
- [ ] Proporcao de pneus, cabine, capo e entre-eixos coerente.
- [ ] Rodado largo Performance BR aprovado visualmente.
- [ ] Lastro frontal com 22 pesos + peso de fixacao modelado.
- [ ] Escapamento inox ASM modelado e integrado ao conjunto.

## High-poly

- [ ] Bordas principais com bevel consistente.
- [ ] Parafusos e tampas em areas proximas da camera.
- [ ] Fundicao do motor e transmissao preparada para bake.
- [ ] Cabine externa e interna separadas.
- [ ] Motor, turbo, common rail, radiador e escape visiveis.
- [ ] ILS separado em partes animaveis.
- [ ] VCRs traseiros com engates rapidos e tampas.
- [ ] Levante traseiro com bracos, cilindros e terceiro ponto separados.

## Rodas e pneus

- [ ] Pneus 800/70R38 traseiros com sulcos profundos.
- [ ] Configuracao dupla traseira completa.
- [ ] Lettering Michelin/Trelleborg modelado ou texturizado.
- [ ] Deformacao sutil no contato com solo.
- [ ] Discos de peso internos e externos nas rodas traseiras.

## Cabine

- [ ] Assento ActiveSeat II com base e giro.
- [ ] Console CommandARM detalhado.
- [ ] Display G5 com tela separada.
- [ ] Volante e coluna separados.
- [ ] Pedais, tapetes e comandos internos suficientes para camera.
- [ ] Vidros separados para shader transparente.

## Iluminacao

- [ ] LED 360 de fabrica completo.
- [ ] Barra LED slim ASM no teto.
- [ ] Farol de milha ASM na grade.
- [ ] Lentes com volume e material proprio.
- [ ] Objetos de luz/emissive separados para configuracao.

## UV e bake

- [ ] UV sem sobreposicao indesejada em areas unicas.
- [ ] Texel density consistente entre carroceria e cabine.
- [ ] High-poly e low-poly nomeados para bake.
- [ ] Normal map sem artefatos visiveis.
- [ ] Metallic/Roughness/Glossiness testados no viewport.
- [ ] Dirt/Wear mask cobrindo areas de uso real.

## Game-ready

- [ ] LOD0 otimizado sem perder silhueta.
- [ ] LOD1 e LOD2 presentes.
- [ ] Collision simplificada.
- [ ] Pivots corretos nos objetos animaveis.
- [ ] Nomes sem acentos, espacos ou caracteres especiais.
- [ ] Materiais agrupados por atlas.

## Giants Editor / FS

- [ ] I3D abre sem erro.
- [ ] Shapes carregam corretamente.
- [ ] Luzes funcionam por grupo.
- [ ] Rodas giram e esterçam corretamente.
- [ ] ILS responde sem clipping visivel.
- [ ] Levante traseiro anima sem atravessar chassi.
- [ ] Mod aparece na loja com imagem e dados corretos.
- [ ] Testado em mapa com lama/sujeira.

## Bloqueio final

- [ ] Passes 01 a 09 documentados.
- [ ] Manifesto em estado `final_modeling_package_locked`.
- [ ] Pasta de exportacao skeleton pronta para ser gerada.
- [ ] Nenhuma alteracao visual nova pendente.
- [ ] Proxima fase limitada a exportacao, XML, textura, teste e empacotamento.

## Aprovacao final ASM

- [ ] Visual robusto e brasileiro.
- [ ] Verde/amarelo/cinza metalico coerentes.
- [ ] Performance BR reconhecivel sem exagero.
- [ ] Interior jogavel em primeira pessoa.
- [ ] Export limpo para FS22.
- [ ] Export pronto para adaptar ao FS25.
