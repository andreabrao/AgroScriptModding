Termos de uso

Estes termos explicam como funcionam os planos mensais, os downloads protegidos e o uso dos mods disponibilizados pela AGRO SCRIPT MODDING.

Assinatura mensal

Cada plano libera uma quantidade mensal de mods para o email usado na compra. O limite e renovado a cada mes enquanto o plano estiver ativo.

O codigo de acesso e pessoal. Compartilhar codigo, token, arquivos ou links privados pode bloquear o acesso da conta.

Downloads digitais

Os mods sao arquivos digitais para Farming Simulator 22. Depois de liberado, o download fica protegido pelo servidor e so abre para assinantes verificados.

A AGRO SCRIPT MODDING pode atualizar, substituir ou remover mods da biblioteca para corrigir problemas, melhorar desempenho ou manter a qualidade do pacote.

Reembolso

Se o pagamento foi feito por engano e nenhum mod foi baixado, o pedido de reembolso pode ser analisado pelo suporte. Quando o acesso ja foi usado, o reembolso depende da avaliacao do caso.

Falhas tecnicas devem ser enviadas ao suporte com email da compra, ID do pagamento e descricao do problema.

Privacidade

O site guarda somente dados necessarios para liberar o plano: email, codigo, plano, ID de pagamento e historico de uso mensal.

Dados de cartao ou Pix sao processados pelo Mercado Pago. A AGRO SCRIPT MODDING nao recebe nem armazena dados bancarios completos.
